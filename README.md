Building and installing Graffophone
==============================


Configuration
-------------

Prerequisites: rust >= 1.73.0, ffmpeg 7, liblilv-dev >= 0.24

Compilation
-----------

    $ cargo build --bin graffophone --release


Execution
---------

    $ cargo run --bin graffophone --release
